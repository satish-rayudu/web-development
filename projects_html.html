Gui calculater:

I developed a GUI calculator using Python's Tkinter library, providing a user-friendly interface for basic arithmetic operations. The design includes standard features like addition, subtraction, multiplication, and division. A standout feature is the addition of a backspace button, allowing users to easily correct input errors. This project showcases practical application of GUI design and functionality enhancements.


Password generator:

I created a Password generator using Python's Tkinter library, offering a simple and intuitive interface for generating secure passwords. The generator allows users to customize password length and character types. A key highlight is the integration of the pyperclip library, which enables users to easily copy generated passwords to the clipboard with a single click. This project demonstrates the effective combination of GUI design and practical utility features.
